# Analysis Tools

This directory contains analysis and diagnostic scripts for the trading bot.

**⚠️ IMPORTANT: These are NOT production code. They are analysis/diagnostic tools only.**

## Scripts

- `analyze_backtest_trades.py` - Analyze backtest trade results
- `analyze_exit_reasons.py` - Analyze trade exit reasons
- `detailed_exit_analysis.py` - Detailed exit analysis
- `trace_immediate_closures.py` - Trace immediate trade closures
- `forensic_analysis.py` - Forensic analysis tools
- `generate_equivalence_report.py` - Generate equivalence reports
- `check_equivalence_progress.py` - Check equivalence test progress
- `run_equivalence_backtest.py` - Run equivalence backtests

## Usage

These scripts are for analysis and debugging purposes only. They do not affect live trading operations.
